== Description ==

EightLaw Lite is free responsive lawyer WordPress theme. It is a clean, minimalist, modern theme ideal for lawyers, law firms and agencies, private attorneys, business consultation, blogging or personal websites. Built on WordPress Live Customizer using cutting-edge technology, it offers you a professional design and powerful support. It is free, yet feature-rich law firm WordPress template. It features full-screen beautiful sliders, multiple sidebar options, team section layouts: grid or list, multiple blog layouts, testimonial section layouts: grid or list, media gallery in carousel slider, contact forms, CTA, social media icons etc. On top of everything, EightLaw Lite is simple and super user-friendly. Demo: https://8degreethemes.com/demo/eightlaw-lite Support forum: support@8degreethemes.com

Install Steps:
--------------

1. Activate the theme
2. Go to the Appearance > Customize page
3. Setup theme options
